Welcome to Game Studio!

Games are developed in Python using Graphical User Interface in CodeSculptor, which is a browser-based Python interpreter, as featured in the online course “An Introduction to Interactive Programming in Python”.

To be deployed directly on brower, the CodeSkulptor-based games are embedded in a JavaScript module built by Dave Alger.

Supported Browers

Games are best view in Chrome 18+, Firefox 11+, and Safari 6+ (CodeSkulptor compatible browsers) even though they may work in Internet Explorer.
